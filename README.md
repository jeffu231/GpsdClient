# GpsdClient
A .NET 8 library for connecting in an easy way to different types of gps devices via GPSD using C#.

The library contains the following ways to connect to gps services:
- Using the GPSD service (not officially supported by GPSD).

The service provides an event-based interface for receiving data from the gps device.

## Example
Included is a simple console application that shows how to use the library.
